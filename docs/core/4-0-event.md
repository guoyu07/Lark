#Lark Core 编程指南 - 处理事件
利用事件处理系统，程序员可以十分方便地响应用户输入和系统事件。Lark事件模型基于文档对象模型 (DOM) 第 3 级事件规范，是业界标准的事件处理体系结构，为 Lark 程序员提供了强大而直观的事件处理工具。    
Lark 事件处理系统与显示列表密切交互。
若要对显示列表有基本的了解，请参考：[显示编程](5-0-display.md)。
